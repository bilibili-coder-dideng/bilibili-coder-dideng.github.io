# Vanction 编程语言教程

欢迎来到Vanction编程语言教程！本教程将从基础开始，逐步介绍Vanction语言的各种特性和用法。

## 1. 环境搭建

### 1.1 安装必要工具

在开始之前，你需要安装以下工具：

- **CMake**：用于构建项目
- **MinGW-w64 GCC**：C++编译器

### 1.2 构建Vanction

按照README.md中的安装步骤构建Vanction编译器：

```bash
git clone https://github.com/bilibili-dideng/VanctionProgrammingLanguage.git
cd VanctionProgramLanguage
mkdir build
cd build
cmake ..
cmake --build .
```

### 1.3 验证安装

运行Hello World示例来验证安装：

```bash
cd ..
./build/Vanction -i examples/test/hello_world.vn
```

如果看到输出"Hello, World!"，说明安装成功。

## 2. 基础语法

### 2.1 Hello World

让我们从经典的Hello World程序开始：

```vanction
func main() {
    System.print("Hello, Vanction!");
    return 0;
}
```

- `func`关键字用于定义函数
- `main()`是程序的入口函数
- `System.print()`用于输出内容
- `return`语句用于返回值

### 2.2 用户输入
使用`System.input()`函数获取用户输入：

```vanction
func main() {
    System.print("请输入你的姓名：");
    var name = System.input();
    System.print("你好，" + name + "!");
    return 0;
}
```

### 2.3 注释

Vanction支持单行注释和多行注释：

```vanction
func main() {
    || 这是一个单行注释
    var a = 10; || 行尾注释
    |* 这是一个
   多行注释 *|
   |\ 这是一个文档注释
   * 用于函数或模块的文档说明
   /|
   """ 这也是一个
   多行注释 """
}

## 3. 变量和类型

### 3.1 变量声明

使用`var`关键字声明变量：

```vanction
var x = 10;          || 整数
var name = "Vanction";  || 字符串
var is_active = true;   || 布尔值
var pi = 3.14159;       || 浮点数
```

### 3.2 常量声明

使用`immut var`声明不可变常量：

```vanction
immut var PI = 3.14159;
immut var VERSION = "1.0.0";
```

常量一旦声明，就不能被修改，否则会引发`ImmutError`。

### 3.3 类型系统

Vanction是动态类型语言，变量类型会根据赋值自动推断：

- **整数**：`10`, `-5`, `0`
- **浮点数**：`3.14`, `-0.5`, `1.0`
- **字符串**：`"hello"`, `'world'`
- **布尔值**：`true`, `false`
- **列表**：`[1, 2, 3]`
- **哈希表**：`{"key": "value"}`
- **对象**：`instance Person("Alice")`

## 4. 函数

### 4.1 函数定义

使用`func`关键字定义函数：

```vanction
func add(a, b) {
    return a + b;
}
```

### 4.2 函数调用

```vanction
func main() {
    var result = add(5, 3);
    System.print("5 + 3 = " + result);
    return 0;
}
```

### 4.3 函数参数

函数可以接受任意数量的参数，参数类型会自动推断：

```vanction
func greet(name, age) {
    System.print("Hello, " + name + "! You are " + age + " years old.");
}
```

### 4.4 返回值

使用`return`语句返回值：

```vanction
func multiply(a, b) {
    return a * b;
}
```

如果没有返回值，函数默认返回`undefined`。

## 5. 控制流

### 5.1 条件语句

#### 5.1.1 if语句

```vanction
func main() {
    var x = 10;
    
    if (x > 5) {
        System.print("x is greater than 5");
    } else-if (x < 5) {
        System.print("x is less than 5");
    } else {
        System.print("x is equal to 5");
    }
    
    return 0;
}
```

#### 5.1.2 逻辑运算符

- `&`：逻辑与
- `|`：逻辑或
- `^`：逻辑异或

```vanction
if (x > 5 & y < 10) {
    || 执行代码
}
```

### 5.3 switch语句

Vanction支持switch语句，用于基于不同条件值执行不同的代码块：

```vanction
func main() {
    var num = 2;
    
    switch (num) {
        case 1 {
            System.print("Case 1 executed");
        }
        case 2 {
            System.print("Case 2 executed");
        }
        case 3 {
            System.print("Case 3 executed");
        }
    }
    
    return 0;
}
```

**说明**：
- `switch`关键字后面跟着要测试的表达式
- `case`关键字后面跟着要匹配的值
- 每个case块用`{}`包围
- 当匹配到某个case时，执行该case的代码块
- 默认情况下，不会自动break，会继续执行下一个case（fallthrough行为）

**支持的类型**：
- 整数
- 字符串
- 布尔值
- 浮点数/双精度数

```vanction
|| 字符串switch
var str = "hello";
switch (str) {
    case "hello" {
        System.print("Hello case executed");
    }
    case "world" {
        System.print("World case executed");
    }
}

|| 布尔值switch
var flag = true;
switch (flag) {
    case true {
        System.print("True case executed");
    }
    case false {
        System.print("False case executed");
    }
}
```

### 5.4 循环结构

#### 5.4.1 while循环

```vanction
func main() {
    var i = 0;
    while (i < 5) {
        System.print("i = " + i);
        i = i + 1;
    }
    return 0;
}
```

#### 5.4.2 do-while循环

```vanction
func main() {
    var i = 0;
    do {
        System.print("i = " + i);
        i = i + 1;
    } while (i < 5);
    return 0;
}
```

#### 5.4.3 for循环

```vanction
func main() {
    for (var i = 0; i < 5; i = i + 1) {
        System.print("i = " + i);
    }
    return 0;
}
```

#### 5.4.4 for-in循环

用于遍历集合：

```vanction
func main() {
    || 遍历列表
    var numbers = [1, 2, 3, 4, 5];
    for (var num in numbers) {
        System.print("Number: " + num);
    }
    
    || 遍历哈希表
    var person = {"name": "Bob", "age": 25};
    for (var key, var value in person) {
        System.print(key + ": " + value);
    }
    
    || 遍历范围
    for (var i in range(5)) {
        System.print("i = " + i);
    }
    
    return 0;
}
```

## 6. 类和对象

### 6.1 类定义

使用`class`关键字定义类：

```vanction
class Person() {
    || 成员变量
    var name;
    var age;
    
    || 构造函数
    instance.init(instance, name, age) {
        instance.name = name;
        instance.age = age;
    }
    
    || 成员方法
    instance.greet(instance) {
        System.print("Hello, my name is " + instance.name);
    }
    
    instance.get_age(instance) {
        return instance.age;
    }
}
```

### 6.2 创建对象

使用`instance`关键字创建对象：

```vanction
func main() {
    var person = instance Person("Alice", 30);
    person.greet();
    System.print("Age: " + person.get_age());
    return 0;
}
```

### 6.3 继承

支持单继承：

```vanction
class Employee(Person) {
    var job_title;
    
    instance.init(instance, name, age, job_title) {
        || 调用父类构造函数
        Person.init(name, age);
        this.job_title = job_title;
    }
    
    instance.work() {
        System.print(this.name + " is working as a " + this.job_title);
    }
}

func main() {
    var emp = instance Employee("Bob", 25, "Software Engineer");
    emp.greet();  || 继承自Person类
    emp.work();   || Employee类自己的方法
    return 0;
}
```

## 7. 错误处理

Vanction使用`try-happen`机制处理错误：

```vanction
func main() {
    try {
        var result = 10 / 0;  || 这会引发DivideByZeroError
        System.print("Result: " + result);
    } happen (DivideByZeroError) as e {
        System.print("Caught an error:");
        System.print("  Text: " + e.text);
        System.print("  Type: " + e.type);
        System.print("  Info: " + e.info);
    }
    
    return 0;
}
```

### 7.1 内置错误类型

- `Error`：所有错误的基类
- `DivideByZeroError`：除零错误
- `ValueError`：值错误
- `TypeError`：类型错误
- `MethodError`：方法错误
- `ImmutError`：常量修改错误
- `RangeError`：范围错误
- `ListIndexError`：列表索引错误

### 7.2 捕获所有错误

使用`Error`类型可以捕获所有错误：

```vanction
try {
    || 可能引发错误的代码
} happen (Error) as e {
    System.print("Caught an error: " + e.text);
}
```

## 8. 数据结构

### 8.1 列表（List）

列表是有序的元素集合：

```vanction
func main() {
    || 创建列表
    var numbers = [1, 2, 3, 4, 5];
    
    || 访问元素
    System.print("First element: " + numbers[0]);
    System.print("Last element: " + numbers[-1]);  || 支持负索引
    
    || 修改元素
    numbers[2] = 10;
    
    || 添加元素
    numbers.add(6);
    
    || 获取列表长度
    System.print("Length: " + numbers.size());
    
    || 遍历列表
    for (var num in numbers) {
        System.print(num);
    }
    
    return 0;
}
```

### 8.2 哈希表（HashMap）

哈希表是键值对集合：

```vanction
func main() {
    || 创建哈希表
    var person = {
        "name": "Alice",
        "age": 30,
        "city": "New York"
    };
    
    || 访问值
    System.print("Name: " + person["name"]);
    
    || 修改值
    person["age"] = 31;
    
    || 添加新的键值对
    person["email"] = "alice@example.com";
    
    || 遍历哈希表
    for (key, value in person) {
        System.print(key + ": " + value);
    }
    
    || 获取所有键
    var keys = person.key();
    System.print("Keys: " + keys);
    
    return 0;
}
```

## 9. 模块和导入

### 9.1 模块系统

Vanction支持将代码组织为模块，便于重用和管理：

```vanction
|| 导入模块
import math;

func main() {
    var result = math.add(5, 3);
    System.print("Result: " + result);
    return 0;
}
```

### 9.2 命名空间

使用命名空间组织代码：

```vanction
|| 定义命名空间
namespace utils {
    func add(a, b) {
        return a + b;
    }
    
    func subtract(a, b) {
        return a - b;
    }
}

func main() {
    var sum = utils:add(5, 3);
    var diff = utils:subtract(5, 3);
    System.print("Sum: " + sum + ", Diff: " + diff);
    return 0;
}
```

## 11. 高级特性

### 11.1 闭包

Vanction支持闭包：

```vanction
func make_counter() {
    var count = 0;
    
    func increment() {
        count = count + 1;
        return count;
    }
    
    return increment;
}

func main() {
    var counter = make_counter();
    System.print(counter());  || 输出: 1
    System.print(counter());  || 输出: 2
    System.print(counter());  || 输出: 3
    return 0;
}
```

### 11.2 Lambda表达式

支持匿名函数：

```vanction
func main() {
    var add = func(a, b) { return a + b; };
    System.print(add(5, 3));  || 输出: 8
    return 0;
}
```

### 11.3 类型转换

```vanction
var num_str = "123";
var num = type.int(num_str);  || 字符串转整数
System.print(num);  || 输出: 123

var float_str = "3.14";
var float_num = type.float(float_str);  || 字符串转浮点数
System.print(float_num);  || 输出: 3.14

var int_num = 42;
var str_num = type.string(int_num);  || 整数转字符串
System.print(str_num);  || 输出: "42"
```
